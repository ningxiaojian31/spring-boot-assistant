# spring-boot-assistant

新华生活小助手项目:仿微博里面的树洞，是一款促进大学生交流的社交软件。

Android端代码：https://github.com/ningxiaojian31/assistant

### 1、该项目作为Spring Boot项目，内置Tomcat，端口为8090，开发工具IntelliJ IDEA

### 2、相关建表请参考sql文件

### 3、数据持久层采用了Spring Data JPA + Mybatis，旨在探索两者的用法的不同，建议一般的查询用JPA，复杂的查询用mybatis,或者使用更加出色的Mybatis-Plus

### 4、不足之处：没加到异常处理机制、返回的json格式也没有做统一格式处理，有兴趣的可以自行增加
